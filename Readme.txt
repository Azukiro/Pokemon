Le projet :
Réalisation du pokémon en java. Comportant différents mode de jeu comme le mode combat aléatoire ou encore le mode League.

Les auteurs :
Courtois Fabien
Billard Lucas

La licence :
GPL (GNU Publique Licence)

Pour ce qui est de compilé ce projet nous vous proposons de passer par le logiciel Eclipse d'ouvrir le dossier du projet de lancer le projet via Eclipse

Dans ce projet nous trouvons une limitations qui oblige le joueur à aller dans l'interface Pokédex et de créer son dresseur, avant de lancer n'importe quel mode de jeu. Mais si vous ne changez jamais de dresseur vous n'aurez qu'à le faire à la première utilisations du jeu car après votre dresseur est sauvegardé.

Certaines Capacités des pokémons ne font pas de dommage mais cela n'est pas indiqué alors ne vous inquiétez pas si une capacité n'inflige pas de dégât à votre adversaire.
